**Messages Friends or /who for a guild invite on login**

### Features

* Automatically checks if player is in a guild on login
* Sends Battle.net whisper to preferred friends (in priority order) requesting guild invite
* Fallback system: searches guild via /who if no preferred friends are online
* In-game GUI for managing settings (/gir config)
* Add/remove/reorder preferred Battle.net friends
* Configurable fallback guild name
* Persistent settings saved between sessions
* Won't spam - only requests once per logout

### Commands

* /gir config - Open settings GUI
* /gir check - Manually trigger guild invite check
* /gir reset - Reset request status (allows requesting again)
* /gir list - List all Battle.net friends with online status
* /gir ver - Show addon version
